# POV
1. ผมเจอคุณลุงวัยเกษียณ มาตีปิงปองที่สวนธนฯ ตอนเย็นเป็นประจำ กับเพื่อนๆหรือนักศึกษา
2. ผมแปลกใจที่ มีอุปกรณ์ไม่เพียงพอ หรือชำรุด
3. หรือมันเป็นเพราะว่า ผู้คนไม่รักษาของ ไม่มีการบำรุงรักษา
4. มันน่าจะดีมากถ้าเราช่วยให้คุณลุงมีอุปกรณ์กีฬา (ปิงปอง) ให้ยืม

## How Might We
+ เราจะทำอย่างไรเพื่อจะ เพิ่มปริมาณและคุณภาพของอุปกรณ์กีฬาปิงปองในสวนธนฯ?
+ เราจะทำอย่างไรเพื่อจะ สร้างระบบบำรุงรักษาอุปกรณ์กีฬาให้มีประสิทธิภาพมากขึ้น?
+ เราจะทำอย่างไรเพื่อจะ สร้างสภาพแวดล้อมที่เอื้อต่อการออกกำลังกายสำหรับผู้สูงอายุในสวนธนฯ?
+ เราจะทำอย่างไรเพื่อจะ ดึงดูดเยาวชนให้มาออกกำลังกายร่วมกับผู้สูงอายุในสวนธนฯ?
+ เราจะทำอย่างไรเพื่อจะ ทำให้การออกกำลังกายในสวนธนฯเป็นประสบการณ์ที่น่าสนใจมากขึ้น?
